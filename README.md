# Google Dino AI Bot

Полный цикл разработки (End-to-End) бота на базе компьютерного зрения и глубокого обучения для автоматизации игры в "Google Динозаврика". 

## 🛠️ Архитектура и стек проекта
* **Сбор датасета (`dataset.py`):** Высокоскоростной захват кадров экрана в реальном времени с помощью библиотеки `mss`..
* **Обучение модели (`model.py`):** Бинарная сверточная нейросеть (CNN) на TensorFlow/Keras. Оптимизирована скорость загрузки данных с использованием `.cache()` и `.prefetch(buffer_size=AUTOTUNE)`. Для предотвращения переобучения применены слои `Dropout`.
* **Инференс (`result.py`):** Запуск обученной модели в продакшн. Оптимизирован вызов графа вычислений через `model(..., training=False)` для достижения максимального FPS. Бот стабильно преодолевает препятствия и набирает ~890 очков.

## 🚀 Ключевые фишки
1. Работа с матрицами и векторами на чистом **NumPy**.
2. Встроенный слой нормализации `Rescaling` внутри архитектуры сети.
3. Оптимизация Windows-потока за счет микро-задержек для стабильного инференса.
